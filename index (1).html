Google Sheets Setup Guide

1
Create a Google Sheet
Go to sheets.google.com → Create a new sheet named OJT Reports. Add these headers in Row 1:
Name | Company | Week | Objectives | Activities | Reflections | Image URLs | Timestamp | Sig_Trainee | Sig_Supervisor | Sig_Coordinator
2
Create a Google Drive Folder
In Google Drive, create a folder named OJT Documentation and note its Folder ID (from the URL: drive.google.com/drive/folders/FOLDER_ID).
3
Open Apps Script
In your Google Sheet → Extensions → Apps Script. Replace the code with the contents of Code.gs (provided in the download). Update the SHEET_ID and FOLDER_ID constants.
4
Deploy as Web App
Click Deploy → New Deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy. Copy the Web App URL.
5
Connect the App
Paste the Web App URL in the Google Sheets Backend panel on the right sidebar and click Connect.
